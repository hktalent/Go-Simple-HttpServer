# Go-Simple-HttpServer
Golang Simple HttpServer


# How Use
```
git clone https://github.com/hktalent/Go-Simple-HttpServer
cd Go-Simple-HttpServer
go build main.go
cd ..

git clone https://github.com/hktalent/Hack-Tools
cd Hack-Tools
yarn build
mv dist ../Go-Simple-HttpServer/static/hacktools

cd ..
git clone https://github.com/hktalent/hackerToolsApp
cd hackerToolsApp/app
yarn build
mv dist ../Go-Simple-HttpServer/static/app

cd ..
git clone https://github.com/hktalent/webssh2
cd webssh2/app
yarn install
yarn start

```
